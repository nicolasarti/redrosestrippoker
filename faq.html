Red Rose Strip Poker
 
the first Strip Poker themed video game series designed to be art
 
100% open source and independent software. Compatible with Windows and Linux PCs.
 
Born with the idea of ​​providing a useful tool for photographers, video makers and models to be able to express their art and even monetize if needed by creating a customized Strip Poker themed video game, protecting their work and their privacy.
 
Maximum confidentiality and the ability to monetize: the contents are encrypted, in addition the author can distribute the game so that an activation key is requested when starting and / or when a certain stage is reached. The activation key will work only on that pc.
 
It is not necessary to install anything to play, just download the archive and unpack it in a folder of your choice which can therefore also be an external hard disk or SSD.
 
Opportunity to monetize: maximum flexibility, any activation key to unlock the game can be provided after receiving payment through any online or non-online tool. For the maximum protection of privacy or to divide the proceeds among several people, cryptocurrencies such as Bitcoin can also be used. A simple bitcoin payment checker is already included in Red Rose Strip Poker.
 
Smart contracts: for maximum privacy and transparency, always taking advantage of cryptocurrencies, a smart contract can also be created to automatically distribute earnings to both the photographer and the model.
 
FAQ:
 
what is Red Rose Strip Poker? it is an opensource video game, developed independently and based on the Kiss Strip Poker Maker project https://github.com/ksge/kspm. You can find the full Red Rose Strip Poker source code here https://github.com/nicolasarti/Red-Rose-Strip-Poker You are free to reuse and modify the source code to create your own strip poker themed video game. I only ask that you do not re-use the Red Rose Strip Poker name and / or the graphics and / or multimedia content published by me.
 
How does Red Rose Strip Poker stand out from other similar video games?
first of all it is not a commercial product and the source code is open and available on github for modification and improvement.
In addition to this, from the gamer's point of view, it is a light weight video game that takes up few resources, it can work without problems even on old PCs or on a VM, it does not have any kind of bloatware. Yes the graphics are not so bad and the media contents can be of very high quality (eg 4K resolution). The multimedia contents with which Red Rose Strip Poker is produced are assembled trying to provide a work of art trying to look to the future with a retrospective eye towards the first strip poker themed video games of the 80s.
 
How can I create my own video strip poker game? I suggest you to learn to use the excellent Kiss Strip Poker Maker project, it is very simple and you can easily get excellent results, once you have learned how to use this engine you can customize the graphics or modify and improve other aspects if you want.
 
I have build my own Strip Poker game, how can I monetize now? you are free to sold it on any platform like ebay, patreon and many more.
 
I am a model or a photographer, I have the multimedia material ready for the game but I am not a programmer and I cannot produce my own version of the video game. How can I do? I can try to help you and I will be happy to cooperate, contact me!
 
I downloaded the game but my antivirus tells me it might be malware, what does that mean? Red Rose Strip Poker does not contain malware, the source code is public, some antivirus some times detect self-produced rare software as potential malware and therefore not very widespread. Inquire with the antivirus support to report a false positive and you will see that in the end you will be confirmed that the game is 100% free from viruses! The more these video games are played in the world, the more the problem will disappear!
 
I have an Apple Mac but I see it is not supported, can't I play? Mac systems are not supported today but you can easilyuse virtualization software such as Parallels, vmware workstation or virtualbox that allow you to install Windows or Linux within the mac os operating system.
 
I would like to play from my smartphone or tablet, is it possible? To date it is not possible and there is no version for smartphone or tablet
 
I would like to play via web browser is it possible? No there is no web based version.
 
What can I do to play this video game while maintaining privacy? No data is ever collected by Red Rose Strip Poker,  furthermore you are free to keep the game on an external encrypted device or to play it inside a virtual machine using for example vmware or virtulbox.
 
I downloaded and played the demo version, I saw that there are no underwear or nude scenes, is there a version where the nude is also included? No, the demo version does not include intimate / nude scenes, try the other complete versions with other models that will be published.
